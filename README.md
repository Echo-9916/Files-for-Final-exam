# Files-for-Final-exam

### 📖 项目简介
本项目由“**钢哥老学长**”在激情地经历了十多门期末考试后有感而发。为了应对考试，作者对部分课程内容进行了总结归纳，主要以科目作为划分，旨在供**天津科技大学（天科大）计算机类**的同僚们传阅使用。希望这些“小玩意”能对各位的备考有所帮助。

### 🌟 核心心血：计算机网络 (Computer Networks)
在本项目涵盖的所有科目中，**计算机网络**凝聚了作者最多的心血，是本仓库的**重磅推荐**资源。

*   **深度总结**：该科目下的 `.md` 总结文件针对考试难点进行了深度复习引导。
*   **多维学习**：推荐配合谢老师(谢希仁)的《计算机网络(第七版)》进行复习，直观理解协议栈及数据传输过程。
*   **即时查阅**：总结文档中已嵌入**超链接**，可直接跳转至对应的详细 PDF 资料。

除计网外，**马克思主义基本原理**与**大学物理 A-2** 的总结内容也具有极高的参考价值。

## 💡 备考复习方法论 (Preparation Methodology)

作为经历了多场“战役”的老学长，我总结了一套针对计算机类考试的快速通关策略，建议配合本仓库资源使用：

### 1. 🔍 三轮复习法
* **第一轮（快速扫盲）：** 打开本项目中的 `.md` 总结文件，结合思维导图快速建立科目框架，明确“哪些是重点”。
* **第二轮（深度攻克）：** 针对 MD 总结中标记的难点，点击超链接跳转到对应的 **PDF 详尽资料**，进行沉浸式阅读。
* **第三轮（实战演练）：** 下载仓库中的历年题或实验报告，查漏补缺，模拟真实考试环境。

### 2. 🧠 计算机网络专项攻略 (Deep Dive)
计网知识点碎、逻辑深，复习时请遵循：**“自顶向下看应用，自底向上看封装”**。
> **秘籍：** <u>把书读厚,再把书读薄</u>。

### 3. 🛠️ 实验与课设指南
对于计组课设和数据结构，不要死记硬背代码。
* 先看仓库里的 **实验报告模版** 理解设计思路。
* 再对照 **Markdown 源码说明** 搞清楚逻辑流转，这样在面对老师提问（验收）时才能游刃有余。
